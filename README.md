# How to Run Locally
==================

1.  **Clone the repository**
2.  **Install dependencies**
    npm install
3.  **Build the application**
    npm run build
4.  **Add the desired platforms to target**
    1.  For iOS
        npx cap add ios
    2.  For Android
        npx cap add android
5.  **Run on the desired platform**
    1.  For iOS
        npx cap run ios
    2.  For Android
        npx cap run android
