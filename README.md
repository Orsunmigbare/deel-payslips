**How to Run Locally**\
**==================**

1.  ****Clone the repository****
2.  ****Install dependencies****
    npm install
3.  ****Build the application****
    npm run build
4.  ****Sync the web assets and configuration with native platforms****
    npx cap sync
5.  ****Run on the desired platform****
    1. For iOS
        npx cap run ios
    2. For Android
        npx cap run android
